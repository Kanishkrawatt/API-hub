GitHub Copilot is an AI pair programmer that provides code suggestions directly in your editor. Powered by OpenAI Codex, it helps write code faster by suggesting entire lines or blocks based on context and comments.

### What you can build

- IDE extensions with AI assistance
- Automated code generation tools
- Developer productivity tools
- Code review assistants
- Documentation generators
- Custom coding workflows
- Team collaboration tools
- Learning and training platforms
