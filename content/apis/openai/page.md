The OpenAI API lets you build products powered by modern AI models. Typical use cases include:

- Assistants and copilots
- Content generation and transformation
- Search and semantic retrieval using embeddings
- Structured extraction and classification

#### What you can build

- A support bot trained on your docs
- A semantic search layer for your product
- A content pipeline that drafts, rewrites, and summarizes
